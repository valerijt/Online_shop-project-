Проект выполнил студент группы Б03-015 Тян Валерий
Проект написан на фреймворке Flask с использованием базы данных sqlalchemy

Чтобы запустить проект необходимо в терминал написать команду:

	flask run
	
Описание проекта: 

Веб-сайт предлагающий пользователю купить одну из трех электрогитар, 
одинаковых по цене и различающихся в особенностях звучания в силу конструкции.
